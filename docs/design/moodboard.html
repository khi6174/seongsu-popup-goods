성수동 팝업을,
집까지 · 외국어로 · 줄 안 서고.

성수동 팝업스토어 공식 굿즈를 한국어·영어로 큐레이션·판매하는 글로벌 쇼핑몰. 외국인 관광객이 3초 안에 "귀엽다, 사고 싶다"를 느끼되, 결제 앞에서는 "믿을 만하다"가 되게.

Playful Pop채택된 디자인 방향
둥근 모서리 · 파스텔 + 포인트스티커/캐릭터 감성
둥근 친근 산세리프영문 가독성 우선

😊 Playful — 감성 영역

둘러보기 · 상품 카드 · 캐릭터 키비주얼. 파스텔 배경 + 스티커 배지 + 마이크로 인터랙션(하트 뿅, 키링 흔들). 여기서 "귀엽다·사고싶다"를 만든다.

↔

🛡️ Trust — 돈·배송·결제 영역

가격 · 배송비 · 총액 · 결제. 한 톤 낮춰 흰 카드 + 진한 잉크(고대비) + "No hidden fees". 여기서 "믿을 만하다"를 만든다.

중심 원칙: 귀여움(Playful) ↔ 신뢰(Trust)의 긴장을 영역별로 분리해서 둘 다 얻는다. 원색을 화면 전체에 쓰면 '장난감 사이트'로 불신받으므로, 포인트색은 버튼/배지에만, 배경은 넉넉한 여백의 뮤트 파스텔.

02 · Color

컬러 팔레트

07 · Guidelines

Do / Don't

Do

  • 파스텔은 배경·큰 면적, 포인트 원톤은 버튼·배지에만.
  • 돈·배송·결제는 흰 카드 + 진한 잉크로 고대비.
  • 가격은 KRW 기준 + 현지통화/USD 병기, 총액만 강조.
  • 언어 토글은 우상단 고정 · 44px 이상 · EN 정확.
  • 품절은 파스텔 유지 + 재입고 알림 + 대체 추천.
  • 마이크로 인터랙션은 감성 영역에만, 짧고 은은하게.

Don't

  • 화면 전체를 원색·고채도로 도배(장난감 느낌·불신).
  • 파스텔만 뿌옇게(대비 없이) → 유아용품처럼 보임.
  • 결제 영역에 파스텔 저대비 텍스트 사용(WCAG 미달).
  • 품절을 회색 도장으로 차갑게 처리.
  • 총액 외 여러 숫자에 강조색 남발(초점 분산).
  • 장식 과한 영문 폰트로 가독성 희생.
