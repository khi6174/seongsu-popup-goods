# 프로젝트 개요 — AI 루키 리그 쇼핑몰 구축

> 이 문서는 프로젝트의 **배경·맥락·목표**를 담습니다. 자주 바뀌지 않는 내용이며,
> 새로 합류하는 사람이나 나중의 나(또는 MCP 지식베이스)가 "왜 이걸 하는가"를
> 가장 먼저 읽는 문서입니다.

## 한 줄 요약
AI 루키 리그 수강생이 **쇼핑몰을 기획 단계부터 직접 구축**하며, 그 과정을 통해
AI 활용과 개발 워크플로우를 학습하는 **공개 학습 프로젝트**.

## 배경 (Context)
- **누가**: "AI 루키 리그" 교육 과정 수강생 (learning-by-building 단계의 학습자)
- **무엇을**: 실제로 동작하고 **오래 유지되는** 온라인 쇼핑몰
- **왜**: 결과물(쇼핑몰) 자체보다, **구축 여정을 통한 AI/개발 개념 학습**이 진짜 목적
- **어떻게**: Claude와 기획 → 디자인 → 개발 → 운영까지 **단계별로 논의**하며 진행
- **특징**: 다른 사람들이 지켜보는 **공개적 성격** → 결정과 사고 과정을 남이 따라올 수
  있게 명확히 기록

## 목표 (Goals)
1. **학습**: 이 프로젝트를 통해 AI 도구 활용, 기획/설계/개발/운영의 전체 흐름을 체득
2. **결과물**: 실제 사용 가능한 쇼핑몰 MVP → 점진적 고도화
3. **자산화**: 의사결정·사고 과정을 문서로 남겨 → 추후 **MCP 지식베이스**로 재활용

## 비목표 (Non-goals) — 지금 단계에서는 하지 않는 것
- 대규모 트래픽 대응, 복잡한 마이크로서비스 아키텍처
- 완벽한 디자인 — 먼저 동작하는 MVP, 다듬기는 나중에
