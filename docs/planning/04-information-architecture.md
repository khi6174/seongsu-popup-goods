# 04 · 정보구조 (Information Architecture, IA)  — v1.0

> 사용자 경험(UX) 개선을 위한 산출물. 화면 목록(`02`)이 "어떤 페이지가 있나"라면,
> IA는 **"그 페이지들과 콘텐츠(팝업·굿즈)를 어떻게 조직하고, 사용자가 어떻게 찾아
> 다니게 할까"** 를 설계한다. 특히 타겟이 외국인 관광객이라, **길 잃지 않고 · 언어
> 장벽 없이 · 원하는 굿즈까지 빠르게** 도달하는 것이 핵심.
>
> **초심자 메모**: IA = "정보의 지도와 이정표". 사이트맵(구조) + 내비게이션(이동 수단)
> + 라벨링(표지판) + 탐색 경로(길)를 정하는 일.

## 1) 조직 체계 (어떻게 묶을 것인가)
콘텐츠를 묶는 기준이 UX를 좌우한다. 두 축을 둔다.
- **1차 축 = 팝업별** (이 몰의 정체성). "OO 팝업 → 그 팝업의 굿즈들". 사용자의 머릿속
  모델("나는 그 팝업 굿즈를 원해")과 일치.
- **2차 축 = 카테고리별** (키링/스티커/에코백…). "특정 팝업이 아니라 그냥 예쁜 키링이
  갖고 싶어" 하는 탐색을 보조.
- **상태 축 = 팝업 진행상태** (열림 / 마감임박 / 지난 팝업). 관광객의 "지금 갈 수 있나?"
  질문에 답.

## 2) 사이트맵 (구조도)
```
홈 (Home)
├── 팝업 둘러보기 (Popups)            ← 1차 축
│     ├── [팝업 상세]  (그 팝업 소개 + 소속 굿즈 목록)
│     │      └── [상품 상세]
│     └── (필터: 열림 / 마감임박 / 지난)
├── 굿즈 카테고리 (Shop by Category)   ← 2차 축 (보조)
│     └── [카테고리별 상품 목록] → [상품 상세]
├── 장바구니 (Cart)
│     └── 체크아웃 → 주문 완료
└── (전역) AI 챗봇 · 언어(KO/EN) · 통화(KRW/USD)
        · 안내(About/배송정보) ← 푸터
```

## 3) 내비게이션 설계
**상단 헤더 (모든 화면 공통)** — 좌→우:
`[로고/홈]  [팝업 둘러보기]  [카테고리]  ────  [🌐 KO/EN]  [₩/＄]  [🛒 장바구니]`
- 외국인 대상이므로 **언어 전환을 헤더 눈에 띄는 위치**에 상시 노출(찾아 헤매지 않게).
- 장바구니 아이콘엔 담긴 개수 배지.

**하단 푸터**: About(우리 몰 소개) · 배송/반품 안내 · 언어 재전환 · (이후) 고객문의.

**전역 위젯**: 우측 하단 **AI 챗봇 버튼**(모든 화면). "무엇을 찾으세요?"를 다국어로.

## 4) 라벨링 (표지판 = 용어)
한/영 라벨을 처음부터 쌍으로 정해 일관성 유지(들쭉날쭉하면 UX 저하).
| 개념 | 한국어 | English |
|------|--------|---------|
| 홈 | 홈 | Home |
| 팝업 목록 | 팝업 둘러보기 | Popups |
| 카테고리 | 굿즈 카테고리 | Shop by Category |
| 팝업 상태 | 열림 / 마감임박 / 종료 | Open / Closing soon / Ended |
| 장바구니 | 장바구니 | Cart |
| 주문 | 주문하기 | Checkout |

## 5) 주요 탐색 경로 (사용자가 굿즈에 도달하는 길)
1. **팝업 기반(메인)**: 홈 → 팝업 둘러보기 → 팝업 상세 → 상품 상세
2. **카테고리 기반(보조)**: 홈/헤더 → 카테고리 → 상품 상세
3. **AI 챗봇 기반(지름길)**: 챗봇에 "지금 갈 만한 팝업?"·"귀여운 키링 추천" → 결과 링크
4. **직접 진입(QR·공유, 2차)**: 팝업/상품 URL로 바로 진입

→ **세 갈래 길**(둘러보기·카테고리·챗봇)이 같은 상품 상세로 수렴하게 설계 =
   사용자 성향(탐색형 vs 목적형 vs 대화형)을 모두 커버.

## 6) UX 개선 포인트 (이 IA가 노리는 것)
- **언어 장벽 최소화**: 언어 전환 상시 노출 + 상품 설명 영어 자동 번역 + 다국어 챗봇.
- **"지금 갈 수 있나?" 즉답**: 팝업 카드에 상태 배지(열림/마감임박)를 크게.
- **관광객 맥락**: 가볍고 부피 작은 굿즈 위주 + 배송 국가/권역을 체크아웃에서 명확히.
- **선택 피로 감소**: 팝업 2~4개로 좁게 큐레이션 → 무한 스크롤 대신 "고른 것만".
- **일관된 이정표**: 라벨(한/영)을 표로 고정해 화면마다 같은 단어 사용.

## 7) 2차 이후 IA 확장 여지
회원(마이페이지·주문내역), 검색창, 위시리스트, 리뷰, 팝업 캘린더 뷰(달력형),
필터 고도화(가격·브랜드), QR 랜딩 페이지.
